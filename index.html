HTML==> HyperText Markup Language

HTML is made up of:

* Text content - what you see
* Markup - what it looks like
* References to other documents - e.g. images or videos
* Links to other pages

alt="text" property is required because both to acknowledge broken requests and to help visually impaired viewers

image tag doesn't have a closing tag because it doesn't have any content

this kind of tags are called "VOID" tags



<br> and <p> Tags

<br> -- break (VOID) -- line break

<p> -- paragraph -- not a void tag, it has a content, therefore needs a closing tag

<br> - inline tag -- which means ending a line

<p> - block tag -- which makes an invisible box around text

inline elements are text


block elements are some space around them
